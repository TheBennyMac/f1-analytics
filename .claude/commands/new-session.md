# /new-session

Use this at the start of any new chat window to orient quickly.

## What this does
1. Read `CLAUDE.md` to reload domain knowledge and architecture patterns
2. Read `BACKLOG.md` to identify the current version and active work
3. Run `git log --oneline -10` to see what was last delivered
4. Summarise in 3 bullet points:
   - What has been built so far
   - What the current active milestone is
   - What the recommended next task is

This replaces the need to re-explain context at the start of every session.
